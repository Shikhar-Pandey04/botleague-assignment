# BotLeague - Frontend UI Assignment

This repository contains the frontend user interface for the BotLeague assignment, built pixel-perfectly from the provided Figma design.

## Tech Stack Used
* React.js
* Tailwind CSS
* Vite

## How to Run Locally
1. Clone this repository.
2. Navigate to the frontend folder: `cd frontend`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
